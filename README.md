crawlmanager
=============================

## Getting Started

### Installation

**Note requires python >= 3.7**

To use crawlmanager, run:
```bash
python setup.py install
```


### Tests

Run tests with:

```bash
py.test --cov ./crawlmanager -vv ./tests/
```
